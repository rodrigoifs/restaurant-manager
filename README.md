# Restaurant Manager 🍕

Uma aplicação simples em linha de comando (CLI) desenvolvida em Python para gerenciamento e cadastro de restaurantes.

## 🔨 Funcionalidades do Projeto

- **Cadastrar restaurantes:** Adiciona novos estabelecimentos com nome e categoria.
- **Listar restaurantes:** Exibe a lista de restaurantes cadastrados de forma organizada e alinhada no terminal.
- **Alternar estado:** Ativa ou desativa o status de um restaurante (Ativo/Desativado).

## 🚀 Tecnologias Utilizadas

- **Python 3**
- Biblioteca nativa `os` para manipulação e limpeza do terminal.

## 🎯 Conceitos Praticados

- Manipulação de dicionários e listas.
- Estruturas condicionais (`if/elif/else`) e de repetição (`for`).
- Tratamento de erros e exceções (`try/except`).
- Formatação de strings (`f-strings` e método `.ljust()`).

1. Certifique-se de ter o Python instalado em sua máquina.
2. Baixe o arquivo `app.py`.
